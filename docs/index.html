EBarimt PosAPI REST API

Built with Go and Fiber

This example demonstrates a fast and efficient REST API for the EBarimt PosAPI using Go language and Fiber framework.

This REST API uses the following dependency: Lambda Platform EBarimt Go Package

Docker Image

A Docker image for this package is available on Docker Hub:

REST API for Ebarimt PosAPI

To pull and run the Docker image, use the following command: 

docker pull munkhaltai/ebarimt:latest
docker run -d -p 3000:3000 --name ebarimt-rest-api munkhaltai/ebarimt

This will pull the Docker image from Docker Hub and run it in a new container named 'ebarimt-rest-api', exposing the API on port 8080.
